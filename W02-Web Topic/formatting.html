แบบฝึกปฏิบัติ: HTML Formatting

เรียนรู้การจัดรูปแบบข้อความด้วยแท็ก HTML พร้อมตัวอย่างและการตกแต่งด้วย CSS

ตัวอย่างผลลัพธ์

  • B: ข้อความตัวหนา
  • I: ข้อความตัวเอียง
  • MARK: ข้อความเน้น
  • SMALL: ข้อความขนาดเล็ก
  • DEL: ข้อความถูกลบ
  • SUP: x2   SUB: H2O

โค้ด HTML

<strong>ข้อความสำคัญ</strong>
<em>ข้อความเน้น</em>
<mark>ข้อความที่ต้องการเน้น</mark>
<small>ข้อความขนาดเล็ก</small>
<del>ข้อความที่ถูกลบ</del>
<sup>2</sup> <sub>2</sub>

กิจกรรม

  • ให้นักศึกษาสร้างหน้าเว็บ 1 หน้า
  • ใช้แท็ก formatting อย่างน้อย 6 รูปแบบ
  • เทียบผลลัพธ์ของ <b> กับ <strong> และ <i> กับ <em>
  • ปรับสี ขนาด และพื้นหลังด้วย CSS

เกร็ดความรู้

แท็กจัดรูปแบบข้อความช่วยให้เนื้อหาสื่อสารเจตนาและความหมายมากขึ้น เช่น <strong> ใช้สำหรับข้อความสำคัญ และ <em> ใช้เน้นความสำคัญหรืออารมณ์
