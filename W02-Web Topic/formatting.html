แบบฝึกปฏิบัติ: HTML Formatting

เรียนรู้การจัดรูปแบบข้อความด้วยแท็ก HTML พร้อมตัวอย่างและการตกแต่งด้วย CSS

🖥 ตัวอย่างผลลัพธ์
B: ข้อความตัวหนา
STRONG: ข้อความสำคัญ
I: ข้อความตัวเอียง
EM: ข้อความเน้นเสียง
U: ข้อความขีดเส้นใต้
MARK: ข้อความที่ถูกเน้น
SMALL: ข้อความขนาดเล็ก
DEL: ข้อความที่ถูกลบ
SUP: x2
SUB: H2O
</> ตัวอย่างโค้ด HTML 
1<b>ข้อความตัวหนา</b>
2<strong>ข้อความสำคัญ</strong>
3<i>ข้อความตัวเอียง</i>
4<em>ข้อความเน้นเสียง</em>
5<u>ข้อความขีดเส้นใต้</u>
6<mark>ข้อความที่ถูกเน้น</mark>
7<small>ข้อความขนาดเล็ก</small>
8<del>ข้อความที่ถูกลบ</del>
9x<sup>2</sup>
10H<sub>2</sub>O
🎯 กิจกรรมฝึกปฏิบัติ
  • ✓ สร้างหน้าเว็บ 1 หน้า
  • ✓ ใช้แท็ก formatting อย่างน้อย 6 รูปแบบ
  • ✓ เปรียบเทียบ <b> กับ <strong> และ <i> กับ <em>
  • ✓ ปรับสีและขนาดข้อความด้วย CSS
  • ✓ บันทึกชื่อไฟล์ว่า formatting.html
💡

เกร็ดความรู้

แท็กจัดรูปแบบข้อความช่วยให้เนื้อหาสื่อสารชัดเจนและมีความหมายมากขึ้น เช่น <strong> ใช้สำหรับข้อความที่สำคัญ และ <em> ใช้สำหรับข้อความที่เน้นความสำคัญหรืออารมณ์ และ <mark> ใช้เน้นข้อความที่ต้องการให้โดดเด่น แท็กเหล่านี้ช่วยเพิ่มความอ่านง่ายและจัดระเบียบเนื้อหาให้ผู้ใช้งานเข้าใจได้ง่ายขี้น
